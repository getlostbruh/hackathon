Hardware & Software

Hardware Components

Arduino Mega

Main controller for the entire system, handling all logic and processing.

Gyroscope

Motion detection sensor used for tracking orientation and movement.

16x16 LED Panels (x4)

Primary display for the maze visualization and game interface.

7-Segment Display

Used to display the code and other numerical information.

Infrared Remote Control

Allows users to set the code

Software Components

Arduino IDE

The Development environment used for programming the Arduino.

Eagle

Used for designing and creating the circuit diagram.

Diagrams.net

Tool for creating flowcharts.

Vscode

Web development platform used to build the hackathon website.
