How it works

Ons project draait rond een interactief doolhof dat bestuurd wordt met een draagbare gyroscoop.

De speler navigeert door het doolhof door het apparaat in verschillende richtingen te kantelen.

Het doolhof is volledig opgebouwd uit 4 16X16 Led Panelen die samen de muren, gangen en de pion (de speler) vormen.

Onder het doolhof bevindt zich een LED-strip die fungeert als een voortgangsbalk.

Deze visualiseert een timer van een vast aantal seconden en creëert zo een gevoel van tijdsdruk.

Bij het succesvol voltooien van een doolhof krijgt de speler een cijfer.

Dit cijfer is onderdeelvan een viercijferige code, die wordt weergegeven op een 7-segmentdisplay boven het doolhof.

Na vier doolhoven heeft de speler de volledige code verzameld.

Deze code wordt vervolgens gebruikt om verder te geraken in de escape room.

De juiste code wordt vooraf ingesteld door de organisator via een infrarood afstandsbediening.

Het volledige systeem draait op één enkele Arduino.

Daarnaast zijn er drie fysieke knoppen voorzien, elk met een unieke functie.

  • Een slow-down knop die de timer vertraagt.
  • Een start/stop knop om het spel te pauzeren of hervatten.
  • Een random kleur knop die van kleur veranderd.
